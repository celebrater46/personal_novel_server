##############################################
使い方
##############################################

サンプルを参考に、「PersonalNovelServer」フォルダの直下に作品名（アルファベット推奨）のフォルダを作ります。

「PersonalNovelServer」直下のリストの「list.txt」は作品名の一覧リストです。
「作品名|フォルダ名」という具合に、間を半角の縦棒で区切って書いていきます。次の作品名を追加するには、改行してください。

各作品フォルダの中にある「list.txt」は各話のサブタイトルを入力します。
「章番号（1から始まる半角数字）|サブタイトル|txtsフォルダ内のファイル名」という具合に、
間を半角の縦棒で区切って書いていきます。次のサブタイトル名を追加するには、改行します。

各作品フォルダの中にある「chapters.txt」には、各章のタイトルを入れます。
第一章「日本編」、とかそんなかんじのやつです。
章題なんかいらねーよって場合は、作らなければ各話のサブタイトルだけ表示されます。

各作品フォルダの中にある「caption.txt」には、作品のあらすじを書きます。

各作品フォルダの中にある「txts」フォルダの中に、各話の.txtファイルを入れます。
「半角数字.txt」という名前にしないとバグります。
頭に余計な 0 を入れて「0001.txt」とかにするとバグります。
途中で数字が飛んでも（1.txt -> 4.txtとか）一応動作しますが、各話リストに貼られるリンクとズレることがあります。

ルビの指定に関しては小説家になろうとかカクヨムとかで使われてる方式が使えます（｜漢字《かんじ》 <- こういうの）。
また、《《》》で傍点が各文字にルビとして追加されます。
よくわかんないという人はとりあえず使わなければ問題ないです。


色々制約がありますが、「ここをもっとこうしたら、便利なのに！」と思いついたそこのあなた。
物臭な作者に言うより、たぶん改変した方が早いです。
